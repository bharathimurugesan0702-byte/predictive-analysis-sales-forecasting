# Predictive Analytics Using Historical Data

## Project Overview
This project uses historical retail sales data to forecast future sales trends using Linear Regression.

## Objectives
- Analyze historical sales data
- Convert sales records into monthly trends
- Build a predictive model
- Forecast future sales

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Google Colab

## Dataset
Superstore Sales Dataset

## Methodology
1. Data Loading
2. Data Preprocessing
3. Monthly Sales Aggregation
4. Linear Regression Model Training
5. Sales Forecasting
6. Data Visualization

## Outcome
Successfully predicted future sales trends using historical sales data and visualized the results through a forecasting graph.
